# Proyecto Alura Store

Este proyecto es un análisis de datos de ventas de cuatro tiendas diferentes. El objetivo es identificar cuál de las tiendas genera menos ganancias y debería ser evaluada para una posible venta.

## Propósito del análisis

El análisis busca responder cuál tienda tiene menor rendimiento, evaluando los siguientes puntos:

- Facturación total
- Ventas por categoría
- Calificación promedio
- Productos más y menos vendidos
- Promedio de envíos realizados por tienda

## Estructura del proyecto

El proyecto contiene los siguientes archivos:

- AluraStoreLatam.ipynb: notebook principal con el análisis completo
- README.md: este archivo con información general

## Ejemplos de resultados obtenidos

- La tienda 3 fue la que menos facturó: $1.156.536.200
- La tienda 3 también presentó productos con pocas ventas
- Aunque fue la mejor evaluada (4.05), no logró un buen desempeño en ventas
- Todas las tiendas tuvieron un promedio similar de envíos

## Cómo ejecutar el notebook

1. Abre el archivo AluraStoreLatam.ipynb en Google Colab o VS Code con Jupyter
2. Ejecuta las celdas paso a paso
3. Revisa la sección final del notebook para ver el informe y la recomendación final
